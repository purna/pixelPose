# PoseForge

A web-based character animator for creating pose animations.

## Features

- Pose characters with adjustable skeleton
- Create and save animations
- Import/export animations as JSON
- Various animation presets (walk cycles, run cycles, etc.)
- Adjustable view (rotate, zoom, perspective)
- Different body types (adult male, female, child, quadruped)
- Onion skinning for animation preview
- Grid and labels for reference

## Getting Started

Simply open `index.html` in a web browser to start using the application.

## Project Structure

- `index.html`: Main HTML file
- `css/`: Stylesheets
- `js/`: JavaScript files
- `data/`: Body definitions and animation presets
- `fonts/`: Font files

## Development

This project uses plain HTML, CSS, and JavaScript. No build steps are required.

Open `index.html` in a browser to run the application.
